# MMF Randomiser

A simple web app for the **Muizenberg Mathematical Fever** game. Players scan QR codes to trigger randomised outcomes for infection, symptoms, and immunity.

## QR Codes

Three QR codes are used in the game, each pointing to a different page:

- **QR1 – Infect:** `/infect/`
- **QR2 – Symptoms:** `/symptoms/`
- **QR3 – Immunity:** `/immunity/`

## Setting up GitHub Pages

1. Go to your repository on GitHub → **Settings** → **Pages**.
2. Under **Source**, select the branch (e.g. `main`) and set the folder to `/ (root)`.
3. Click **Save**. GitHub will publish the site and provide a URL (e.g. `https://username.github.io/mmf-randomiser/`).
4. Update your QR codes to point to the relevant sub-pages on that URL.
